2. Background

Measuring joint angles is essential in physical therapy and rehabilitation. Traditional goniometers are manual, slow, and often inconsistent. This project aims to improve accuracy and usability through a digital Bluetooth-enabled goniometer using the BNO085 IMU.

3. Design

The goniometer is built around the BNO085 sensor, capable of delivering fused rotation vectors with high accuracy. An ESP32 microcontroller reads the IMU data and transmits joint angles via BLE to a smartphone app. An OLED display is included for real-time feedback, and the user can press a button to capture reference angles.

The device is powered by a 3.7V LiPo battery.
